# Классификация ландшафтов с использованием нейронных сетей

## Постановка задачи

Цель проекта - разработать нейросетевое решение для классификации различных типов ландшафтов на основе их изображений, при этом отдельные части изображений заменены не несущими информацию фрагментами. Данная задача имеет практическую ценность, например, для различных беспилотных систем, от которых требуется распознавать местность в условиях плохой видимости.

## Формат входных и выходных данных

Входные данные: RGB-изображение ландшафта (формат .JPG, размер изображения 256 х 256). Выходные данные: JSON-словарь, содержащий предсказание класса и уверенность модели в своем предсказании.
